Mouse Over Event

The mouseover event occurs when the mouse pointer is over the selected element.
The mouseover() method triggers the mouseover event, or attaches a function to run when a mouseover event occurs.
(We want when we move the mouse over the image an alert to pop up telling us about the image).

Mouse Out

The mouseout event occurs when the mouse pointer leaves the selected element.
The mouseout() method triggers the mouseout event, or attaches a function to run when a mouseout event occurs.
(We want when we move the mouse out the paragraph under the heading mouse out an alert to pop up telling us about the image).

Hover Event

The hover() method specifies two functions to run when the mouse pointer hovers over the selected elements. This method triggers both the mouseenter and mouseleave events.
Note: If only one function is specified, it will be run for both the mouseenter and mouseleave events.

Fade In Event

The fadeIn() method gradually changes the opacity, for selected elements, from hidden to visible (fading effect).
Note: Hidden elements will not be displayed at all (no longer affects the layout of the page). Tip: This method is often used together with the fadeOut() method.
(We want when we click heading the image below to fade in slowly).
